# Machine Vision Class Notes - IIUM

Personal notes and implementations for the Machine Vision course at IIUM.

## Overview

This repository contains code examples, assignments, and experiments from my Machine Vision class. Feel free to explore and use as reference for your own learning.

## Environment Setup

### System Specifications

- **GPU**: NVIDIA GeForce RTX 3060 Laptop GPU
- **CUDA**: 12.6
- **CuDNN**: 9.8

### Software Requirements

- **Python**: 3.10.18
- **PyTorch**: 2.9.0+cu126
- **OpenCV**: 4.13.0-dev

### Installation

1. **Create a virtual environment** (recommended):

```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
```

OR

```bash
   conda create --name Vission_Env python=3.10.18
   conda activate Vission_Env 
   conda deactivate Vission_Env 

```

2. **Install dependencies**:

```bash
   pip install -r requirements.txt
```
